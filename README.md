# vagrant-debian-bookworm64-docker

## 要件

- Vagrant 2.4.1+
- VirtualBox 7.0+

## インストール

```bash
git clone https://github.com/kumarstack55/vagrant-debian-bookworm64-docker.git
Set-Location .\vagrant-debian-bookworm64-docker

#vagrant destroy -f

#vagrant box list
#vagrant box update --box debian/bookworm64

vagrant up
```
